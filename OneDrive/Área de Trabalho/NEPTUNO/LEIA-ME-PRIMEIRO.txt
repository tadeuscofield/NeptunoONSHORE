⚠️ ATENÇÃO - LEIA ISTO PRIMEIRO!
----------------------------------

Para o sistema funcionar SEM ERROS, você PRECISA usar
um servidor HTTP local.

❌ NÃO FUNCIONA: Duplo clique em index.html
✅ FUNCIONA: Usar SERVIDOR.bat

Por quê? Navegadores bloqueiam arquivos JSX por segurança
(política CORS). Servidor HTTP resolve isso!

🚀 COMO COMEÇAR (MÉTODO CORRETO):
----------------------------------

1. Duplo clique em: SERVIDOR.bat

2. Aguarde aparecer no terminal:
   "Acesse: http://localhost:8080"

3. Abra o navegador e vá para:
   http://localhost:8080

4. Use o sistema normalmente!

5. Quando terminar:
   - Volte no terminal
   - Pressione Ctrl+C
   - Confirme com S (Sim)

🐛 PROBLEMAS COMUNS:
--------------------

Problema: Tela em branco, erro CORS
Solução: Use SERVIDOR.bat ao invés de duplo clique

Problema: "Python não reconhecido"
Solução: Instale Python ou Node.js

Problema: Porta 8080 em uso
Solução: Feche outros servidores ou use porta diferente
